# EZmissions - Emissions Testing Website

A modern, high-performance, and SEO-optimized website for EZmissions, providing fast drive-thru emissions testing in Sandy Springs and Alpharetta, Georgia.

## 🚀 Features

- **Local SEO Optimized**: Targeted keywords and metadata for Sandy Springs and Alpharetta.
- **JSON-LD Schema**: Structured data for Local Business (AutoRepair) to improve search engine visibility.
- **Vercel Analytics**: Built-in tracking for visitor insights.
- **Responsive Design**: Fully mobile-friendly layout with modern aesthetics.
- **Interactive UI**: Smooth scroll animations, sticky navigation, and a dynamic hero section.

## 🛠 Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom vanilla styling with CSS variables.
- **JavaScript**: Client-side interactivity and animations.
- **Vercel Analytics**: For performance and visitor tracking.

## 📂 Project Structure

- `index.html`: Main entry point with SEO and Schema markup.
- `styles.css`: Custom design system and layout styling.
- `script.js`: Interactive components and animations.
- `charger.png`: High-quality info section image.
- `clean-sky.png`: Background image for the hero section.

## 💻 Local Development

To run the website locally, you can use any static file server. For example, using Python:

1. Open your terminal in the project directory.
2. Run the following command:
   ```bash
   python3 -m http.server 8080
   ```
3. Open your browser and navigate to `http://localhost:8080`.

## 🌐 SEO Best Practices

This site implements the following SEO strategies:
- Targeted meta tags for local service areas.
- Semantic HTML tags (`header`, `main`, `section`, `footer`).
- Image alt tags for accessibility and image search optimization.
- JSON-LD for rich snippets in Google Search results.
